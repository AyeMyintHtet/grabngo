# GrabNGo

This project contains everything you need to run your app locally.

## About

GrabNGo is a comprehensive delivery application designed to bring everything you need right to your doorstep. Whether it's a hot meal from your favorite restaurant, fresh groceries for the week, essential pharmacy items, or a package that needs sending, GrabNGo handles it all with speed and reliability.

### Key Features

*   **Multi-Service Delivery**: Seamlessly order Food, Groceries, and Pharmacy items, or send Packages across the city.
*   **Fast Delivery**: Experience rapid delivery times with an average of 15 minutes.
*   **Live Tracking**: Track your orders in real-time with our integrated map features.
*   **User-Centric Design**: A modern, responsive interface built for ease of use.

## Prerequisites

1.  Clone the repository using the project's Git URL
2.  Navigate to the project directory
3.  Install dependencies: `npm install`
4.  Create an `.env.local` file and set the right environment variables (if any)

## Running the App

Run the app in development mode:

```bash
npm run dev
```

## Building for Production

To build the app for production:

```bash
npm run build
```

## Linting

To run the linter:

```bash
npm run lint
```
